# Employee Payroll System

A small object-oriented Python project that calculates employee salaries using class-based design.

---

## Features
- Supports full-time and part-time employees
- Salary calculation using polymorphism
- Clean class hierarchy
- Easy to extend with new employee types

---

## Concepts Used
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation

---
